# Hospital Patient Management System

A simple console-based Hospital Management System built using Python.

## Features
- Add Patient
- View Patients
- Search Patient
- Update Patient
- Delete Patient
- Data stored in CSV file

## Concepts Used
- OOP (Object-Oriented Programming)
- File Handling
- CSV File Storage
- CRUD Operations
